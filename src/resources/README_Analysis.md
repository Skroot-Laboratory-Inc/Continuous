# Analysis File Descriptions
This file contains important information about the files present in the Analysis directory generated at experiment end.
## Files present
All files present are related to data handling. Information generated by the Application and information entered by the user.
### Experiment Summary.pdf
This is a pdf formatted results document, used to see a quick snapshot of the experiment results.
- The objective use of this file is to see the results of the experiment, in the same format as the UI.
- The first page is intended to be a quick visual of the experiment holistically.
### Experiment Summary.csv
This contains the raw time in hours and SGI for each reader used during the experiment.
- Data from this file is populated using the smooth analyzed files from each Reader.
- The SGI shown is the result of smoothing the raw curve before analyzing it for the peak.
- This data can be plotted in Excel to generate the plots seen during the experiments, in the UI
### Setup Form.png
This file is for user reference. It can be used to see what options were selected when the experiment was started.
### Reader {readerNumber} Second Axis.csv
This file contains all information that the user chose to enter during the experiment, and the time it was entered.
- One file will be present for each reader used during the experiment, that had information entered for the second axis.
